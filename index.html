Predicting Rock Properties Using Log Data (Fall 2025)

Project Stack: Python, TensorFlow, Kaggle

Training a small neural net to predict rock properties such as porosity and permeability from well log data. The dataset I am training the neural net on is rougly 8000 lines of well log data from Kansas Geological Survey. This dataset contain data on resistivity, spontaneous potential, gamma ray, bulk density, neutron and density porosity data.

The aim is to use typical well log data like neutron and gamma ray to predict more complex rock properties like porosity and permeability that are more difficult and expensive to measure directly. By training a neural network on this data, I hope to be able to accurately predict these rock properties from the more easily obtainable well log data.

Pet Rescue: Autonomous Robot (Summer 2025)

Project Stack: C++, C, Python, KiCad, OnShape, ESP32

My ENPH 253 team (Luca Jones, Harchetan Chohan, Taichi Kamei) designed and built an autonomous rescue robot capable of navigating a complex obstacle course, identifying and retrieving ‚Äúpet‚Äù targets, and returning them safely to the home zone, all without human control.

The brain of our robot was an ESP32 microcontroller, chosen for its power and versatility. A custom state machine was built to manage the various modes of operation such as line following and pet detection.

The navigation of the robot relies on infrared LEDs and phototransistors to detect the line of electrical tape that traces throughout the entire course. By reading which phototransistors are on or off, the robot can determine its relative position to the line and make adjustments using PID control to stay on course.

To identify and rescue the pets, our robot utilizes 2D time-of-flight sensors to detect the presence of pet targets. To grab the pets, our robot uses a custom planar 2-bar linkage arm and a 4-bar claw to pick up the pet and store it in the bucket on board the robot. The arm and claw were built using 3D printing, laser cutting, and water-jet cutting.

To lift the bucket and hook it to the zipline, we designed a continuous cascade mechanism that uses a DC motor to wind a rope around a spool, lifting the bucket vertically.

Document Translator (Winter 2025)

Project Stack: Python, Google Translate, Google Cloud Vision, Google Cloud Stroage

During my coop at Condor Energies, I developed a pipleine that automates the process of translating documents from Uzbek, Russian and Kazakh to English. For this project I had to handle a wide variety of document types such as scanned PDFs, Word, Excel and PowerPoint files. Each had its own challenges when it came to extracting text as well as writing the translated text back into the original format.

Scanned PDFs

This involved using Optical Character Recognition (OCR) to extract text from scanned documents such as old drilling reports and well logs. Then using the Google translate API to translate the extracted text before finally writing the translated text back into a new PDF that closely resembled the original document. A particular challenge with this task was ensuring the formatting and layout of the original document was preserved in the translated version. For most PDFs I was dealing with they were scanned documents, so in order to write the translated text back to the PDF I had to 'remember' the pixel location of each word in the original document and then write the translated text back to those same locations.

Production Forecasting Automation Tool (Winter 2025)

Project Stack: Microsoft VBA, Arps Equation

During my coop at Condor Energies, I developed an automation tool to streamline the production forecasting process for natural gas wells. The tool ingests historical production data, cleans and preprocesses it, and then applies decline curve analysis to forecast future production rates. The tool generates field-level and well-level forecasts, complete with visualizations and summary reports. The tool successfully reduced about 3 days worth of work for a resevoir engineer down to just a few hours once per month.
